## 这次最重要的变化

### 1. 不登录也能直接试用

老版本中，未登录用户点击“立即开始编辑”会进入登录页。新版本现在支持游客编辑器：

- 打开首页；
- 点击“立即开始编辑”；
- 直接进入编辑器体验。

游客每天可以免费导出 3 次图片。登录后可继续完整使用，并享受无限制导出和云端保存能力。

### 2. 首页更直观

新版首页直接展示成品谱面预览，让你一眼看到最终可以生成什么样的陶笛谱。品牌、按钮、示例输出和功能说明也都重新整理，更适合新用户快速理解。

**邮件配图位 2：成品谱面导出效果**

建议插入图片：`promo-export-score-result.png`  
图片内容：“故乡的原风景片段 1=G 4/4”完整成品谱面。  
图片说明：新版不只提供编辑能力，更强调清晰、可打印、可分享的最终谱面输出。

### 3. 编辑器 V2 更顺手

新版编辑器采用全新的 V2 工作台：

- 左侧元素面板更清晰；
- 右侧谱面预览更紧凑；
- 顶部工具栏支持保存、导出、返回乐谱列表、撤销/恢复；
- 可以选择是否显示速度；
- 支持歌词、指法图、演奏表达等常用编辑能力。

**邮件配图位 3：V2 编辑器工作台**

建议插入图片：`promo-editor-v2-workspace.png`  
图片内容：左侧工具面板、右侧谱面预览、顶部工具栏。  
图片说明：新版编辑器把输入、预览、保存和导出放在同一个更清晰的工作台里。

### 4. 谱面操作更符合直觉

这次还优化了不少细节：

- 添加延长线、小节线、反复线后，不会打断连续输入；
- 选中元素后按 Del 可以直接删除；
- 删除元素时，相关圆滑线和合并时值线会自动取消；
- 圆滑线可以跨过延长线、休止符和小节线连续显示；
- 八分、十六分、三十二分休止符也可以参与合并时值线；
- G 调 `♭7` 指法映射已修正，更符合实际使用习惯。

• 最简化且效果最稳的操作方式是：

  Supabase 只负责导出注册用户邮箱，Resend 负责群发。

  不要让网站接口一次性给所有用户发，这类任务容易遇到 Vercel 超时、重复发送、失败不可追踪、退订合规问题。

  推荐流程

  1. 在 Supabase SQL Editor 查询已验证邮箱：

  select
    id,
    email,
    created_at
  from auth.users
  where email is not null
    and email_confirmed_at is not null
  order by created_at desc;

  然后导出 CSV。

  2. 在 Resend 中导入联系人
     进入 Resend 的 Contacts / Audiences，把 CSV 导入进去。
  3. 用 Resend Broadcasts 创建更新推荐邮件
     你可以使用之前项目根目录里的邮件文案：

  NEW_VERSION_EMAIL_COPY.md

  建议邮件里包含：

  - 新版首页和编辑器截图
  - “立即体验新版”按钮：https://www.ocarinana.com
  - 一句退订说明，例如：“如果你不希望继续收到产品更新邮件，可以回复本邮件退订。”

  4. 先发测试邮件
     先发给你自己的 QQ、Gmail、Outlook 邮箱各一封，确认：

  - 中文正常显示
  - 图片正常加载
  - 按钮链接正确
  - 没进垃圾箱

  5. 再正式群发
     如果用户数量不多，Resend Broadcasts 是最省事的方式。
