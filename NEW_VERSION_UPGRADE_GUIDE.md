# Ocarinana 新版本升级推广指南

适用场景：官网公告、产品更新页、用户邮件、社群发布、社交媒体推广。  
对比基准：老版本 `main` 分支 vs 当前新版本 `feature/editor-v2-refactor`。

## 一句话推荐

新版 Ocarinana 从“在线陶笛谱生成工具”升级为“可直接试用、体验更完整、输出更专业的陶笛谱编辑器”。首页更适合商业推广，编辑器更像正式生产工具，游客可以免登录体验，注册用户则获得更完整的保存与导出能力。

## 核心升级摘要

| 模块 | 老版本 | 新版本推荐 |
|---|---|---|
| 首页 | 传统左右分栏 Hero，图片展示偏基础 | 轻量商业化首页，品牌更明确，直接展示成品谱面输出 |
| CTA 路径 | 未登录点击“立即开始编辑”进入登录页 | 未登录直接进入游客编辑器 `/editor`，降低试用门槛 |
| 编辑器架构 | 旧 iframe 编辑器链路，体验割裂 | React V2 原生编辑器，工具栏、面板、谱面统一 |
| 编辑器视觉 | 操作区和谱面密度偏传统 | 更紧凑、更清晰、更像商用品质的工作台 |
| 音符输入 | 部分元素插入后容易打断连续输入 | 音符、延长线、小节线、反复线输入更连贯 |
| 删除能力 | 缺少直接键盘删除体验 | 选中元素后按 Del 删除，并自动取消相关圆滑线/时值线 |
| 圆滑线/时值线 | 遇到延长线、休止符时视觉或操作不完整 | 圆滑线跨延长线/休止符连续显示，休止符也可参与时值线合并 |
| 指法映射 | G 调 b7、高中低音区存在映射误差 | 修正 G 调 b7：中音区以 ♭7 替代 7，并更新低音可用范围 |
| 速度显示 | 必须显示速度，最低 40 | 可选择是否显示速度信息，更符合不同谱面习惯 |
| 游客试用 | 未登录用户无法直接进入编辑体验 | 游客可直接编辑，未登录用户每日最多导出 3 次图片 |
| 登录用户权益 | 编辑和保存路径不够直观 | 登录后可创建 V2 乐谱、云端保存、无限制导出 |

## 首页升级亮点

### 1. 更适合商业推广的第一屏

老版本首页偏工具介绍，视觉上更像一个功能说明页。新版本首页改为浅色纸感背景、品牌色 CTA、真实成品谱面预览，更容易让用户在第一屏理解：

- 这是陶笛谱工具；
- 可以生成带指法图的成品谱；
- 可以直接开始编辑；
- 输出结果适合练习、教学、打印和分享。

### 2. 品牌辨识度增强

品牌标识 `Ocarinana` 改回更有辨识度的 `Spicy Rice` 字体，导航视觉更鲜明。对于商业推广来说，这比普通系统字体更容易形成记忆点。

### 3. 示例输出更直接

首页示例区不再用“模拟简谱 + 指法图”的拼接展示，而是直接展示成品谱面图片：

`示例输出：故乡的原风景片段 1=G 4/4`

这样用户能直接看到最终导出效果，减少理解成本。

**推荐配图：** 使用 `promo-export-score-result.png`。  
**图片文案建议：** “不只是编辑音符，而是直接生成可分享的陶笛谱成品。”

### 4. CTA 更适合转化

老版本未登录点击“立即开始编辑”会跳转登录页，容易造成流失。新版本未登录点击后直接进入游客编辑器，先体验再注册，更适合投放、社群转化和自然流量承接。

## 编辑器样式升级

### 1. 从旧 iframe 体验升级为 V2 原生编辑器

老版本主要通过 `/protected/scores` 旧编辑器链路加载 iframe 编辑器。新版本新增 `/protected/editor/v2`，由 React 组件直接组成：

- 顶部工具栏；
- 左侧元素面板；
- 右侧谱面画布；
- 自动保存；
- 图片导出；
- 撤销/恢复；
- 返回“我的乐谱”。

整体体验更统一，也更利于后续继续扩展商业功能。

**推荐配图：** 使用 `promo-editor-v2-workspace.png`。  
**图片文案建议：** “左侧选择元素，右侧实时预览，一屏完成陶笛谱制作。”

### 2. 谱面更紧凑，阅读效率更高

新版本压缩了谱面标题区、小节间距、音符内部行距、歌词行、圆滑线槽、表情符号行和“添加小节”按钮留白。视觉上更像正式谱面，不会显得松散。

### 3. 工具栏更像实际工作台

新工具栏支持：

- 返回我的乐谱；
- 保存；
- 导出图片；
- 编辑标题；
- 选择调号：C / F / G；
- 选择拍号：2/4、3/4、4/4、6/8；
- 可选显示速度；
- 撤销 / 恢复。

速度信息可关闭，适合不需要标注 BPM 的简谱场景。

### 4. 面板分区更清晰

左侧元素面板按使用习惯组织：

- 指法图显示/隐藏；
- 歌词显示/隐藏与清空；
- 高音 / 中音 / 低音；
- 时值；
- 休止符；
- 音符修饰；
- 小节 / 反复；
- 演奏表达。

这比老版本更适合新用户学习，也更适合长期用户高频使用。

## 编辑器交互升级

### 1. 连续输入更顺畅

老版本或旧逻辑里，添加延长线、小节线、反复线后，对应元素可能被选中。用户随后点击数字音符时，可能误替换刚添加的元素。  
新版本调整为：添加延长线、小节线、反复线后不自动选中该元素，后续可以继续输入音符。

### 2. 支持键盘 Del 删除

新版本支持选中谱面元素后按 `Delete` 键删除。删除时：

- 只删除当前选中元素；
- 后续元素自动前移；
- 与该元素相关的圆滑线、合并时值线会取消；
- 用户可根据需要重新添加连接。

这个行为更符合常见编辑器直觉。

### 3. 圆滑线显示更完整

新版修复了两个音符之间存在延长线、休止符、小节线时，圆滑线轨迹中断的问题。现在圆滑线会跨越中间元素连续显示，视觉上更像真实乐谱。

### 4. 休止符可参与时值线合并

八分、十六分、三十二分休止符本质是 `0` 加时值符号。新版本允许这些休止符与其他音符一起参与合并时值线，解决了旧版本里休止符无法合并的限制。

### 5. G 调指法映射更准确

新版本根据实际指法规则修正 G 调：

- 高音区移除 b7；
- 中音区以 `♭7` 替代原有 `7`；
- G 调低音区允许 2、3、4、5、6、7；
- G 调中音 `♭7` 映射 `/webfile/static/G-graph/b7.webp`。

谱面显示也从 `b7` 优化为更符合简谱习惯的 `♭7`。

### 6. 可控制速度信息显示

有些谱面不需要标注速度。新版本新增“显示速度”开关：

- 开启：显示 `♩ BPM`；
- 关闭：谱面顶部不显示速度；
- 关闭不会清空 BPM，之后可重新开启。

## 游客试用规则升级

### 老版本规则

未登录用户点击“立即开始编辑”会进入登录页。用户必须注册或登录后才能真正进入编辑体验。

### 新版本规则

未登录用户可以直接进入 `/editor` 进行游客试用：

- 可直接编辑谱面；
- 可预览指法图；
- 可导出图片；
- 每个未登录游客每天最多导出 3 次图片；
- 超过限制后提示注册/登录；
- 登录用户不受每日 3 次限制，可继续完整使用。

### 推荐对外表达

“无需注册即可体验编辑器；游客每天可免费导出 3 次，登录后可无限制导出并保存到云端。”

这个规则适合商业推广：既降低首次体验门槛，又为注册转化提供自然理由。

**推荐配图：** 使用 `promo-homepage-hero.png`。  
**图片文案建议：** “无需注册，先试做一张陶笛谱；登录后继续完整创作。”

## 对外 FAQ

**Q：新版是否仍然免费？**  
A：可以免费试用。游客每天可导出 3 次图片，登录后可以继续完整使用。

**Q：不登录可以用吗？**  
A：可以。点击首页“立即开始编辑”即可进入游客编辑器。

**Q：为什么游客导出有限制？**  
A：为了保证服务稳定，同时让新用户可以先体验核心能力。需要长期制作陶笛谱的用户建议登录使用。

**Q：旧乐谱还能打开吗？**  
A：乐谱列表会根据乐谱版本进入对应编辑器，新版创建的乐谱会进入 V2 编辑器。

**Q：新版适合哪些用户？**  
A：陶笛学习者、音乐老师、社群谱面整理者、内容创作者。
